# bus_booking_system

这是数据库原理课程的大作业

### 开发环境：

​	python3.6.3 + Django2.1.1

### 实现的功能：

​	**所有用户：** 注册，登录

​	**用户：** 选车，选座位

​	**司机：** 检票

​	**公司雇员：** 查看用户信息
